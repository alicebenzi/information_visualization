Project Dataset

Description

  • The data set contains quality related factors pertaining to red and white variants of the Portuguese "Vinho Verde" wine.
  • There are 1599 instances of red wine and 4898 instances of white wine
  • There are 11 input features per variant that helps describe the quality of wine. All the 11 attributes are numerical in nature.
  • There is one qualitative variable that indicates a quality score for a particular combination of the input feature set

License Information (Source)

  • P. Cortez, A. Cerdeira, F. Almeida, T. Matos and J. Reis. Modeling wine preferences by data mining from physicochemical properties. In Decision Support Systems, Elsevier, 47(4):547-553, 2009.

Motivation

  • Wine producers constantly brag about the quality ratings that their wines receive from critics, because a high wine rating — implying high quality — translates into increased sales for a wine. However most of the times we are hardly aware of the underlying factors that make a wine superior. Being a wine enthusiast, I became fairly interested in the data set and would love to see the patterns that emerge from the data. The data can be viewed on my github repo here for red and white.
