Project Sketch

Planned Tools

  • The data set contains 11 quality related factors that determine the quality score pertaining to red and white variants of the Portuguese "Vinho Verde" wine.
  • I plan to use D3 for my final visualisation.
  • I plan to use Python (pandas dataframe) or R to cleanse the data, deal with missing values, aggregate/scale data if necessary for the various plots.

Planned Techniques

  • Scatterplot matrix - I plan to obtain a scatter plot matrix of quality score plus all the 11 variables to see interactions between variables. There will be separate scatter plot matrices for both the Red and White variants. A scatter plot matrix should give me a sense of the variables that deem important for each of the variables to obtain a quality score. The points will be color coded according to the variables.
  • Parallel coordinate plot - A parallel coordinate plot will also be obtained. The plot will include both the variants and I plan to have different colors (red and golden if suitable) for the red and white variants. This will help distinguish the factors that improve the quality of red wine vs that of white. For the parallel coordinate plot, I will include only those variables that help discover a pattern between red and white. The variables will be decided based on the scatterplot matrix and trying different variations of the parallel coordinate plot.
  • Small multiple bar charts - For each of the 11 variables, small multiple bar plots will be obtained. There will be 11 small multiple bars per variant. I plan to have red and white variants side by side. For example, the variable "residual sugar" will have a bar corresponding to each quality score i.e. for how much residual sugar did white wines with quality score 1 contain on an average.

Planned Interaction

  • Scatterplot matrix - I plan to implement brushing here that will help choose a variable and look for its trends. I will try to link the interactions across the scatterplot matrices of the two variables.
  • Parallel coordinate plot - I plan to implement brushing here again. The brushed area will be highlighted which will help filtering patterns as well
  • Small multiple bar charts- When hovered on any of the bars, bars of that particular quality score will be highlighted across all multiples. A tool tip with details on the percentage quantity will also show up on hovering
